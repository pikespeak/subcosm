<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Reddit platform → trigger handler | Untrusted/extra-field trigger payloads cross into the server here |
| Reddit post iframe → engine render | The webview boots Phaser/WebGL in an embedded mobile iframe (capability not guaranteed) |
| client webview → own server | Same-origin `/api` only; no client-supplied sub id is trusted |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | onPostCreate/onCommentCreate payload | mitigate | Tolerant `.passthrough()` Zod schema with typed required ids; parse at the boundary; tolerate/ignore malformed fields, never trust raw (V5) |
| T-03-02 | Spoofing/Info disclosure | sub id source | mitigate | Derive `sub` from `context.subredditId` server-side; ignore any client-supplied sub (V4) |
| T-03-03 | Denial of Service | WebGL context in mobile iframe unavailable | accept (this plan) | Spike confirms the Canvas2D fallback behind the same Scene seam; verdict recorded — full render path gated on this outcome |
| T-03-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages this phase (RESEARCH Package Legitimacy Audit: all four Devvit/Hono packages pre-existing first-party); slopcheck N/A |
</threat_model>
