1. **Seed demo history (D-01, SUB-02):** backfill the demo subreddit with the deterministic 30-day simulator arc so judges open onto a deep, populated "depth = time" universe instead of an empty day-1. Both are real, mod-gated, KEPT features (D-08) — not temporary scaffolding.

Purpose: make the demo post populated + controllable so judging (primarily on the demo link) lands, and give the folded UAT a trigger it can fire on demand.
Output: backfill module + two mod-menu endpoints + devvit.json menu items, all boundary-parsed and reusing existing tested paths.
